# Videos de Demostración - Nordicol

Este directorio contiene los videos de demostración del Software Nordicol.

## Archivos requeridos:

### Video principal:
- `nordicol-demo.mp4` - Video de demostración en formato MP4 (formato principal)
- `nordicol-demo.webm` - Video de demostración en formato WebM (formato alternativo para mejor compatibilidad)

### Imagen de portada:
- `../images/nordicol-video-poster.jpg` - Imagen de portada que se muestra antes de reproducir el video

## Especificaciones recomendadas:

### Video:
- **Duración**: 2-5 minutos máximo
- **Resolución**: 1920x1080 (Full HD) o superior
- **Formato**: MP4 (H.264) y WebM para mejor compatibilidad
- **Tamaño**: Optimizado para web (< 50MB recomendado)
- **Audio**: Incluir audio explicativo o música de fondo

### Imagen de portada:
- **Formato**: JPG o PNG
- **Resolución**: Misma que el video
- **Tamaño**: < 500KB
- **Contenido**: Captura representativa del video con texto "Play" o logo de Nordicol

## Notas:
- El video se reproduce automáticamente con controles nativos del navegador
- Se incluye fallback para navegadores que no soportan HTML5 video
- La imagen de portada se muestra mientras el video se carga
